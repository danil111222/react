// ФУНКЦИЯ useState
Каждый компонент обладает каким-то состоянием в REACT. При изменении этого состояния REACT понимает что произошли изменения в состоянии
и перерисовывает объект 

Пример записи:

const [likes, setLikes] = useState(5) likes - это состояние, setLikes - функция которая изменяет состояние

//ХУКИ - это функции REACT, начинаются со слова use. ХУКИ МОЖНО ИСПОЛЬЗОВАТЬ ТОЛЬКО НА ВЕРХНЕМ УРОВНЕ ВЛОЖЕННОСТИ
( ТО ЕСТЬ НЕ МОЖЕМ ВКЛАДЫВАТЬ ХУКИ В УСЛОВИЯ В ЦИКЛЫ И ТД.)
